Need help with SealAPI? Choose the channel that fits your needs.

## Email support

For general questions, integration help, and account inquiries:

**[support@sealmyidea.com](mailto:support@sealmyidea.com)**

We aim to respond within **24 hours** on business days (Monday to Friday, CET timezone).

## Reporting an issue

Include as much of the following as possible for faster resolution:

| Information       | Example                                |
| ----------------- | -------------------------------------- |
| API token prefix  | `tok_abc1...` (first 8 characters only)|
| Endpoint          | `POST /api/seal/public/seal-my-data`   |
| HTTP status code  | `400`, `401`, `500`                    |
| Error message     | `"Invalid regulation value"`           |
| Request timestamp | `2026-07-16T10:30:00Z` (with timezone) |
| File size         | `2.4 MB`                               |
| Regulation        | `ZertES` or `eIDAS`                    |

<Warning>
  Never share your full API token in emails or support tickets. We will never
  ask for your complete token; the first 8 characters are enough for
  identification.
</Warning>
